# Customer Analytics: Response Time Distribution by Support Channel

## Email
23f3000228@ds.study.iitm.ac.in

## Project Overview
Heller Schamberger and Rau customer experience analysis for retail client support efficiency.

### Business Context
Analysis of customer support response times across different channels to identify efficiency opportunities and optimize resource allocation for executive decision-making.

### Support Channels Analyzed
- **Email Support**: Traditional email-based customer service
- **Live Chat**: Real-time chat support
- **Phone Support**: Voice call support
- **Social Media**: Social platform customer service

### Key Metrics
- Response time distributions
- Channel efficiency comparisons
- Service level analysis

### Visualization
- **Chart Type**: Violin plot showing response time distribution density
- **Dimensions**: 512x512 pixels
- **Libraries**: Seaborn for professional styling
- **Purpose**: Executive presentations and strategic planning

### Files
- `README.md`: Project documentation
- `chart.py`: Python script generating the visualization
- `chart.png`: Professional violin plot visualization

### Business Value
This analysis helps in:
- Resource allocation optimization
- Service level agreement planning
- Customer experience improvement
- Support channel strategy development
